# NavSumaro 🎓
**One platform. Your entire campus.**

[cite_start]NavSumaro is an integrated academic platform designed specifically for the Naga College Foundation community[cite: 235, 237]. [cite_start]It serves as a unified hub for students to collaborate, exchange resources safely, and find academic help, eliminating the need for fragmented social media groups[cite: 303, 359].

## 🚀 Features

- [cite_start]**Institutional Verification**: Secure registration restricted to `.edu.ph` email addresses to ensure a trusted student-only environment[cite: 362, 396].
- [cite_start]**Academic Forums & Q&A**: A dedicated space for subject-specific help with upvoting and reputation systems[cite: 376, 391].
- [cite_start]**Student Marketplace**: A safe, verified space to buy and sell textbooks, uniforms, and school supplies[cite: 379, 400].
- [cite_start]**Real-time Messaging**: Direct and group chat capabilities for seamless project coordination[cite: 370, 392].
- **Campus Events & Trending Topics**: Stay updated with what's happening around the NCF campus in real-time.

## 🛠️ Tech Stack

- **Frontend**: React with Vite & TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React / React Icons
- **Backend**: Node.js & Express
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: Google OAuth 2.0
